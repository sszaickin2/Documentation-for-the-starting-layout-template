Работа с шаблонизатором PUG

Категории: Документация "Стартового шаблона"

Для того чтобы работать с PUG достаточно просто создать в корне папки src файлы .PUG, например index.pug. Сборка сама переключится на работу с .PUG файлами вместо .HTML файлов.
Внимание! Добавлять подключение app.js и style.min.css в файлы .PUG не нужно, это делается автоматически.
